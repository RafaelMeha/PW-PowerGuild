3. How Information is Used:

Describe the methods used to collect information, such as through user input, cookies, and tracking technologies.

4. Purpose of Collection:

Specify the reasons for collecting user information, such as for account creation, payment processing, and improving the streaming service.

5. User Controls:

Explain how users can access, modify, or delete their personal information and account settings.

6. Third-Party Sharing:

Clarify if and how user information is shared with third parties, such as advertisers or analytics services. If applicable, mention any third-party plugins or integrations.

7. Security Measures:

Detail the security measures in place to protect user information from unauthorized access or disclosure..

8. Data Retention:

Specify how long you will retain user data and the criteria used to determine the retention period.
